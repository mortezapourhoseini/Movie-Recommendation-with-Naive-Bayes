# Movie Recommendation with Naïve Bayes

This repository contains two Jupyter notebooks demonstrating movie recommendation using the Naïve Bayes algorithm. The notebooks use the MovieLens datasets for training and evaluation.

## Files
`Movie-Recommendation-with-Naïve-Bayes.ipynb` : Main notebook for movie recommendation.

## Dataset
`ml-32m/` or `ml-latest-small/` : [MovieLens datasets](https://grouplens.org/datasets/movielens/).

## Usage
download your appropriate dataset from the site.

Open the notebook in Jupyter and and update data_path 

run the cells to see the recommendation workflow.
